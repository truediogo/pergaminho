O que são Embeddings?

Entendendo Embeddings

Uma explicação visual e interativa sobre como computadores transformam palavras em números — e por que isso muda tudo.

1

Computadores não entendem palavras

Nós humanos lemos frases e entendemos significados instantaneamente. Mas computadores só trabalham com números. Como fazer uma máquina "entender" que "cachorro" e "cão" significam a mesma coisa?

cachorro inteligência amor programação felicidade natureza saudade ciência viagem música
? 01001000 01110101 01101000
Como representar
significado com bits?
2

A grande ideia: vetores de significado

Um embedding é uma função que transforma texto em uma lista de números decimais — chamada de vetor. Cada número captura uma "fatia" do significado da frase. Um modelo como o MiniLM gera vetores com 384 dimensões!

"O gato dormiu no sofá"
Modelo de Embedding
0.234 −0.891 0.456 −0.123 0.678 −0.345 0.567 …384d
3

Cada dimensão captura algo

Imagine (de forma simplificada) que cada número do vetor mede uma "qualidade" do texto. Quanto mais alto o valor, mais presente aquela qualidade está.

Para a frase "O gato dormiu no sofá", o modelo poderia (conceitualmente) atribuir valores altos para dimensões ligadas a "animal", "descanso" e "lar" — e valores baixos para "tecnologia" ou "esporte".

Na prática, as 384 dimensões não têm nomes — o modelo aprende sozinho quais padrões de linguagem representar. Mas o princípio é o mesmo: significados parecidos geram números parecidos.

Medindo a semelhança

A distância entre dois vetores nos diz o quão parecidos são os textos. Usamos a similaridade de cosseno — quanto mais próximo de 1, mais parecidos.
